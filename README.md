# ASKERP Marketing Website

Static bilingual (Arabic / English) landing page for ASKERP.

## Preview locally

Open `index.html` in a browser, or run a simple server:

```powershell
cd website
python -m http.server 8080
```

Then open http://localhost:8080

## Deploy free (GitHub Pages)

1. Push the `website/` folder to GitHub (or use a `gh-pages` branch with only this folder).
2. In the repository: **Settings → Pages → Source** → deploy from branch / folder `website`.
3. Your site will be at `https://<username>.github.io/<repo>/` (or a custom domain).

## Deploy free (Netlify)

1. Drag the `website` folder to [Netlify Drop](https://app.netlify.com/drop), or connect the repo.
2. Publish directory: `website`.
